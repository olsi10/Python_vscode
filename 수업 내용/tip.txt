클래스에서 멤버 메소드끼리 호출할 때도 앞에 self.을 붙여야 한다

메서드 내에서 다른 메서드 호출 (주관식)

isinstance()
- 특정 클래스의 인스턴스 객체인지 아닌지 확인
- 어떤 클래스로부터 만들어진 객체인지 확인
(파이썬은 bool 값이 첫 번째)

클래스 변수 -> 자바의 클래스 변수 (static)와 같다
- 공통으로 사용하는 변수
- 파이썬은 static 안 씀

멤버 변수 -> self가 붙은 구조
클래스 변수 -> 클래스명이 붙은 구조

@classmethod
- 클래스 메소드를 정의할 때 사용하기
- 첫 번째 인자가 cls (클래스의 정보를 담은 변수)
- cls 값을 통해 클래스 관련 정보에 접근 가능
- ****클래스 변수를 클래스 메소드 안에서 사용할 때 cls 라고 써야함

@classmethod ======= 구멍 뚫은 문제 출제
